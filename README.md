# ДЗ-6 Устиченко М.Я.
Выполнение ДЗ-6 Устиченко М.Я.

Код взят из ДЗ-1 Для выполнения ДЗ-6

Код "Нахождение средних букв" определяет среднюю часть слова.

Он вычисляет длину строки, находит её середину и:

- если количество букв нечётное - выводит одну среднюю букву;
- если количество букв чётное - выводит две средние буквы.

Код работает для слов любой длины и автоматически определяет, какой вариант вывести

Код "Списки парней и девушек" формирует пары из списков парней и девушек.

Он проверяет, совпадает ли количество элементов в списках.

Если длины различаются, выводится предупреждение, что кто-то может остаться без пары.

Затем оба списка сортируются в алфавитном порядке, и программа выводит идеальные пары, сопоставляя имена в отсортированных списках.
